Projetos

To-Do-List

Criação de uma To-Do-List com uso de Javascript puro, criando novas tarefas a partir da entrada de texto do usuário. Edição do status e do conteúdo da tarefa, exclusão de tarefas e mudança de ordem de preferências com arraste. Há também uma edição full-stack desse projeto no GitHub abaixo.

Carrinho de Compras

Projeto de adição de produtos em um carrinho de compras, com funcionalidades de adição de produtos por um botão de adição, e a remoção de produtos com um botão de subtração (quando chega em 0 ele exclui o produto do carrinho).

Jogo da Adivinhação

Projeto de criação de um jogo de adivinhação, com o objetivo de treinamento da utilização de eventos DOM e obtenção de valores dados pelo usuário

Intro JS-DOM

Atividade de introdução à linguagem Javascript com a criação de um contador, usando o DOM (Document Object Model), pegando as referências das tags e classes HTML e as modificando em Js.

Intro CSS

Projeto de introdução à linguagem CSS. Uso de estilos em cascada, fontes, referências com classes e tags, estilização de forms (select, input e button) e criação de cards com display flex.

Intro HTML

Projeto de introdução ao conceitos da linguagem HTML. Uso de títulos, parágrafos, links, imagens e listas.
